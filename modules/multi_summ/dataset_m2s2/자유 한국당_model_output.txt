문재 이 ㄴ 대통령 이 지나 ㄴ 27 일 ( 현지 시간 ) 김대중 전 대통령 의 아들 이 ㄴ 태 전 공사 의 강연 을 적대 적 행위 로 표현 하 었 다고 보도 하 었 다 . 이 에 대 하 어 문 대통령 의 삼남 이 ㄴ 김정 은 국방 위원 회 제 1 위원 장 은 이날 트위터 에 게재 하 ㄴ 발언 을 트위터 에 올리 ㄴ 것 으로 전하 어 지 었 다 ( 사진 ) .
문 대통령 에 대 하 ㄴ 여론 조사 에서 문재 이 ㄴ 대통령 의 국정 수행 지지도 가 역대 최대 를 기록 하 었 다 . 김 시장 은 지나 ㄴ 4 월 30 일 전국 성인 2503 명 을 대상 으로 조사 하 ㄴ 결과 ( 1 % ) 신뢰 수준 이 라는 시각 에 는 송철호 % 포인트 오르 ㄴ 32.4 % 로 나타나 았 다 . 이 는 지난 해 12 월 24 ~ 26 일 집계 되 ㄴ 울산 경찰 청장 의 집계 에서 1,2 % 포인트 내려가 ㄴ 것 이 다 .
한국 당 은 지나 ㄴ 4 월 지방 선거 를 앞두 고 홍 대표 의 혜안 에 대 하 ㄴ 지지 를 호소 하 었 다 . 홍 대표 는 이날 오전 페이스북 을 통하 어 김정은 의 해임 이 필요 하 다고 주장 하 었 고 홍준표 자유 한국당 대표 는 색깔 론 공세 를 잇 어 가 았 다 .
홍 대표 는 19 일 ( 현지 시간 ) 페이스북 을 통하 어 김정은 의 요구 만 들 어 주 고 지지 하 ㄴ다는 칼럼 을 작성 하 ㄴ 것 으로 알려지 었 다 . 이 에 자유 한국당 은 조선일보 를 협박 하 어 조선일보 가 협박 하 는 것 을 보 고 한 ㆍ 미 연합 훈련 을 취소 하 고 미군 철수 도 안 하 ㄴ다고 하 었 다는 주장 이 다 .
한국 당 은 10 일 ( 현지 시간 ) 페이스북 을 통하 어 비 핵 화 와 북 핵 문제 를 해결 하 ㄹ 수 있 는 계기 가 되 어야 하 ㄴ다고 강조 하 었 다 . 홍 대표 는 10 일 오전 ( 현지 시각 ) 논평 을 내 고 북중 정상 회담 을 앞두 고 홍 대표 의 위장 평화 정책 을 비판 하 는 목소 리 를 높이 ㄴ 것 이 다 .
자유 한국당 홍준표 대표 가 13 일 전체 회의 를 열 고 태 전 공사 의 강연 을 적대 적 행위 로 표현 하 었 다 . 이 에 대 하 어 한국 당 은 이 를 이행 하 는 것 은 절대 불가능 하 ㄹ 것 이 라고 주장 하 었 으나 한국 당 도 즉각 반발 하 고 있 다 .
문재 이 ㄴ 대통령 과 김정 은 북한 국방 위원 회 제 1 위원 장 이 지나 ㄴ 28 일 ( 현지 시간 ) 남북 정상 회담 을 개최 하 기 로 합의 하 었 다고 밝히 었 다 . 홍 대표 는 이날 자신 의 페이스북 을 통하 어 김정은 의 양손 에 선물 보 따리 는 물론 무기 까지 들리 어 보내 어 주 ㄴ 것 이 라고 주장 하 었 고 홍준표 자유 한국당 대표 는 판문점 선언 문 에 잇 어 판문점 선언 을 하 지 않 을 것 으로 보이 ㄴ다 .
홍 대표 는 27 일 ( 현지 시간 ) 남북 정상 회담 에 대 하 어 “ 한반도 문제 의 해법 은 국가 의 문제 이 라는 시각 이 있 다 ” 고 밝히 었 다 . 홍 대표 와 홍준표 대표 는 이날 논평 을 나 의 페이스북 을 통하 어 한반도 비 핵 화 와 북 핵 문제 를 해결 하 ㄹ 수 있 는 계기 가 되 ㄹ 가능 성 이 있 다고 판단 하 고 있 다 .
홍 대표 는 지나 ㄴ 27 일 ( 현지 시간 ) 호주 멜버른 의 ABC 방송 에서 첫 머리 로 남북 정상 회담 을 보도 하 었 다 . 홍 대표 의 막말 에 대 하 어 “ 안 후보 는 민주당 과 한국 당 을 동시 에 지지 하 는 것 은 안 되 ㄴ다 ” 고 말 하 ㄴ 것 으로 전하 어 지 ㄴ 것 이 다 .
김 후보 는 20 일 국회 의원 회관 에서 열리 ㄴ 기자 회견 에서 “ 자유 민주 주의 시장 경제 신념 이 확실 시 되 고 있 다 ” 고 말 하 었 다 . 안 후보 는 안 후보 의 단일 화 관련 언급 에 대 하 어 박 시장 을 비판 하 ㄴ 후 기자 들 과 만나 ㄴ 것 이 다 . 김 후보 의 보수 정체 성 관련 언급 은 안 후보 가 당선 되 었 고 안 지방 선거 에 는 박정희 전 대통령 과 같 은 이름 의 후보 자 도 있 었 으며 이 후보 자 는 사 % 의 득표 율 로 1 위 에 오르 았 다 .
